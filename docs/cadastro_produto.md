# Cadastro de Produto

Cada produto deve ser classificado de acordo com sua categoria.
 - Bebidas: Refrigerantes, Cervejas, Sucos e Vinhos;
 - Perfumaria: Desodorantes, Sabonetes, creme dental, Shampoo e Condicionador
 - Limpeza: Detergente, Água sanitária, Desisfetante, Sabão em barra.

 - Produto
    - Nome da categoria (ex: Bebidas)
    - Código do produto;
    - Marca do produto;
    - produto;
    - SKU;
    - Preço;
    - Quantidade;
    - Validade; 
    - Data de Entrada.

- Validade do Produto
    - Quantidade
    - Validade
    - Data de Entrada
